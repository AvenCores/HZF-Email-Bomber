# Помощник по скачиванию
* Скачиваем Download Manager.py
* Открываем
* Нажимаем на кнопку "Скачать Email Bomber V1.0"
* Переходим в директорию "C:\Bomber"

* Скачиваем архив
* Распаковываем в удобную нам директорию
* Запускаем Terminal
* Переходим в корень с бомбером
